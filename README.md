# Zoover reviews client

> Project was built using react-create-app.

## Installing

```
$ npm install 
```

## Usage example

> reviews-api should be started before client app.

```
$ npm start
$ npm test
```

## Docker 

```
// Build docker image
$ docker build -t zoover/reviews-client .
$ docker run -p 9090:80 zoover/reviews-client
```
